# 🚀 TalentMeet

TalentMeet is a full-stack MERN-based interview platform designed for real-time coding practice and technical interviews. It allows hosts to create coding sessions and participants to join, collaborate, and solve problems in an interactive environment.

---

## 🌟 Features

- 🧑‍💻 **VSCode-Powered Code Editor**  
  Real-time coding experience with a modern editor interface.

- 🔐 **Authentication with Clerk**  
  Secure and seamless user authentication.

- 🎥 **1-on-1 Video Interview Rooms**  
  Conduct live coding interviews with video communication.

- 🧭 **Dashboard with Live Stats**  
  Track performance, sessions, and activity.

- 🔊 **Media Controls**  
  Toggle mic, camera, screen sharing, and recording.

- 💬 **Real-time Chat Messaging**  
  Communicate instantly during sessions.

- ⚙️ **Secure Code Execution**  
  Run code in an isolated environment for safety.

- 🎯 **Auto Feedback System**  
  Get instant results based on test cases (Success/Fail).

- 🎉 **Interactive UI Feedback**  
  Confetti on success and notifications on failure.

- 🧩 **Practice Problems (Solo Mode)**  
  Solve coding problems independently.

- 🔒 **Room Locking System**  
  Restricts rooms to only 2 participants.

- 🧠 **Background Jobs with Inngest**  
  Handles async operations efficiently.

- 🧰 **REST API (Node.js + Express)**  
  Backend APIs for all functionalities.

- ⚡ **Data Fetching with TanStack Query**  
  Efficient state management and caching.
